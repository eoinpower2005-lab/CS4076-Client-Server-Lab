# Lab 4: TCP Client-Server Application

This project contains four Java files which demonstrate IP Addresses and TCP client-server communication and networking

### 1. IPFinder.java
prompts you to enter a hostname and returns its IP address

### 2. IPAddrDemo.java
automatically displays the IP address of your local machine

### 3. TCPEchoServer.java and TCPEchoClient.java
These two programs must be run together. The server is run first, then run the client. Separate terminal windows should open for each.
Enter a message in the client terminal and it will be displayed in the server terminal and echoed back to the client. Running multiple instances 
of TCPEchoClient should be individual and not interfere with eachother. To disconnect a client, type quit in the terminal. The server and client communicate over port 1234.
